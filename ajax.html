AJAX

AJAX permite que una página web se comunique con el servidor sin recargar. Usa JavaScript, el objeto XMLHttpRequest, y el DOM para actualizar solo lo necesario.

Pasos:

  1. Se genera un evento
  2. JavaScript crea un XMLHttpRequest
  3. Se envía la solicitud
  4. Se recibe la respuesta
  5. Se procesa y actualiza la vista
const xhr = new XMLHttpRequest();
xhr.open('GET', 'https://api.ejemplo.com/datos', true);
xhr.onload = () => {
  if (xhr.status === 200) console.log(JSON.parse(xhr.responseText));
};
xhr.send();

Ventajas: mejora la velocidad, experiencia de usuario e interactividad.

Desventajas: requiere JS activado, afecta el SEO, puede ser complejo y vulnerable si no se controla.
