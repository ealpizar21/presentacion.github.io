JSON y XML

JSON: Formato ligero y legible, ampliamente usado en APIs. 

{
  "nombre": "Carlos",
  "edad": 30
}

XML: Más estructurado, menos común hoy en día.


  Carlos
  30

JavaScript: Usamos JSON.stringify() para codificar y JSON.parse() para decodificar. 

// Codificar
let persona = { nombre: "Ana", edad: 25 };
let json = JSON.stringify(persona);

// Decodificar
let jsonText = '{"nombre":"Ana","edad":25}';
let objeto = JSON.parse(jsonText);
