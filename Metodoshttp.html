Métodos HTTP

  • GET: Obtener datos
  • POST: Crear datos nuevos
  • PUT: Actualizar completamente un recurso
  • PATCH: Modificar parcialmente un recurso
  • DELETE: Eliminar un recurso

Ejemplo GET:

GET /usuarios/5

Respuesta:
{
  "id": 5,
  "nombre": "Ana",
  "edad": 30
}

Ejemplo POST:

POST /usuarios
Enviado:
{
  "nombre": "Luis",
  "edad": 25
}
Respuesta:
{
  "id": 6,
  "nombre": "Luis",
  "edad": 25
}

Ejemplo PUT:

PUT /usuarios/5
Enviado:
{
  "nombre": "Ana María",
  "edad": 31
}
Respuesta:
{
  "id": 5,
  "nombre": "Ana María",
  "edad": 31
}

Ejemplo PATCH:

PATCH /usuarios/5
Enviado:
{
  "edad": 32
}
Respuesta:
{
  "id": 5,
  "nombre": "Ana María",
  "edad": 32
}

Ejemplo DELETE:

DELETE /usuarios/5
Respuesta:
{
  "mensaje": "Usuario eliminado con éxito"
}

Códigos de respuesta comunes:

  • 200 OK, 201 Created
  • 400 Bad Request
  • 401 Unauthorized
  • 404 Not Found
  • Internal Server Error
