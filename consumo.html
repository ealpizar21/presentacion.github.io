Consumo del API REST con JavaScript

XMLHttpRequest:

const xhr = new XMLHttpRequest();
xhr.open('GET','https://api.ejemplo.com/items', true);
xhr.onload = () => {
  if (xhr.status === 200) console.log(JSON.parse(xhr.responseText));
};
xhr.send();

Fetch API:

fetch('https://api.ejemplo.com/items')
  .then(r => r.ok ? r.json() : Promise.reject(r.status))
  .then(data => console.log(data))
  .catch(e => console.error(e));

jQuery AJAX:

$.getJSON('https://api.ejemplo.com/items')
  .done(data => console.log(data))
  .fail((_, s, e) => console.error(s, e));
