¿Qué es una API?

Una API permite la comunicación entre diferentes aplicaciones. Actúa como un puente, conectando sistemas para el intercambio de datos sin intervención del usuario. 

{
  "app": "Clima",
  "consulta": "https://api.clima.com/datos?ciudad=SanJose"
}
